## 사용 기술
  + Android kotlin
  + Coroutine
  + Retrofit2
  + MVP Structure
  + Google Firebase (Auth, Firestore)
  + Naver Map Api
  + Jetpack Navigation

## 앱 개요

 > 코로나 학번이 학교에 오지 못하고 3학년이 되어 가고 있습니다.  
 코시국 아닐때는 선배가 신입생 밥사주면서 맛집도 알려주고 그러는 거라던데..  
 코로나로 인해 그러지 못해서, 주변 밥집 정보를 접하지 못한 분들을 위해.  
 학교 주변 식당 정보 공유 앱을 만들어보면 어떨까 생각해서. 
 실습겸 만들고 있습니다.
